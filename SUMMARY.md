## ✨ 核心功能亮點

### 1. 法規諮詢（Legal Intelligence）
- ✅ 精準引用《都市更新條例》具體條文

  - 危老獎勵
- ✅ 分配比例試算
- ✅ 財務可行性評估
- ✅ 危老 vs 都更完整比較

- ✅ 中立顧問立場（地主/住戶/建商）
- ✅ 同意門檻達成策略
- ✅ 談判技巧指導
- ✅ 爭議處理建議
- ✅ 風險管理方案

### 4. 法律諮商（Legal Counsel）
- ✅ 代為拆除程序說明
- ✅ 少數不同意戶權益保障
- ✅ 信託機制建議

### 5. 文書製作（Documentation）

- ✅ 會議紀錄、送審文件

## 🎯 技能特色

### 專業性
- 🏛️ **精準法條引用** - 所有法律解釋都包含具體條文編號
- 📊 **結構化輸出** - 使用統一格式（📋現況分析、⚖️法規適用、💡專業建議、⚠️風險提醒、📝後續步驟）
- 🗺️ **在地化考量** - 涵蓋台北市、新北市等地方自治法規

### 語言品質
- 🇹🇼 **純繁體中文** - 無簡體字混用
- 📖 **專業術語** - 使用台灣都更業界標準用語
- 💬 **清晰表達** - 條列式、表格化呈現複雜資訊

### 實用性
- 🎓 **多種使用場景** - 地主、住戶、建商、更新會、建築師、律師
- 📝 **可執行建議** - 每個回應都包含具體後續步驟
- ⚖️ **法律免責** - 明確標示資訊僅供參考，不構成正式法律意見

## 🚀 立即開始使用

### 最快速的方法

```bash
# 1. 進入專案目錄
cd /Users/jerryliu/UrbanRenewSkill

# 2. 啟動 Claude Code
claude-code

# 3. 開始提問（範例）
```

**提問範例：**
```
我們社區在台北市信義區，想推動都更需要多少住戶同意？
```

```
請比較權利變換和協議合建的優缺點

```

```bash
cd ~/any-project

```
