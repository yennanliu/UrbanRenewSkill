## 範例集合

本文件提供台灣都市更新專家技能的典型使用範例，展示不同場景下的問答互動。

## 範例 1：同意門檻查詢

### 用戶提問
```
我們社區在台北市大安區，屬於劃定更新地區內，想推動都更需要多少住戶同意？
```

### 預期回應
```markdown
## 劃定更新地區都更同意門檻說明

### 📋 現況分析
您的社區位於台北市大安區劃定更新地區內，適用較寬鬆的同意門檻標準。

### ⚖️ 法規適用

依據《都市更新條例》第22條第1項規定，劃定更新地區內需符合以下四項門檻：

| 同意項目 | 法定門檻 |
|---------|---------|
| 土地所有權人數 | 超過 1/2 |
| 土地及建物所有權人數 | 超過 1/2 |
| 土地總面積 | 超過 1/2 |
| 合法建物總面積 | 超過 1/2 |

**重要：** 以上四項標準需「同時」滿足，缺一不可。

### 💡 專業建議

1. **先進行產權調查**
   - 清查所有土地及建物所有權人名冊
   - 確認各戶持分比例及建物面積
   - 注意繼承未辦理登記的特殊案例

2. **計算策略**
   - 建議以「面積」為優先攻克目標（通常較易達成）
   - 針對大面積持有者進行重點溝通
   - 預留安全空間，建議達成率至少60%以上

3. **溝通技巧**
   - 舉辦社區說明會，說明更新效益
   - 提供試算表，讓住戶了解自身權益
   - 建立LINE群組或社區網站，即時回應疑問

### ⚠️ 風險提醒

- **共有產權計算**：如有多人共有一筆土地，需個別計算同意比例
- **建物合法性**：僅「合法建物」面積納入計算，違建不計入
- **同意書效力**：同意書需載明同意範圍，避免日後爭議
- **撤回限制**：依第22條第5項，簽署同意後不得任意撤回

### 📝 後續步驟

1. 委託專業團隊進行產權調查（建議找都更顧問公司）
2. 製作初步意願調查表，評估達成可行性
3. 若可行，開始正式募集同意書
4. 達成門檻後，向台北市都更處提出事業計畫申請

---

**相關法規：**
- 都市更新條例第22條（同意門檻）
- 台北市都市更新自治條例第7條（劃定更新地區程序）
```

## 範例 2：容積獎勵評估

### 用戶提問
```
我們這棟建築物想申請都更容積獎勵，請問綠建築獎勵和智慧建築獎勵可以同時申請嗎？最多可以拿到多少容積？
```

### 預期回應
```markdown
## 都更容積獎勵疊加試算

### 📋 現況分析
綠建築獎勵與智慧建築獎勵可以同時申請並疊加，但需注意總獎勵上限。

### ⚖️ 法規適用

#### 台北市容積獎勵上限（依台北市都市更新自治條例第15條）

| 獎勵類別 | 獎勵幅度 | 取得條件 |
|---------|---------|---------|
| **基本獎勵** | 最高 30% | 更新計畫內容品質 |
| **綠建築獎勵** | 10% | 取得綠建築標章（銀級以上）|
| **智慧建築獎勵** | 10% | 取得智慧建築標章（銀級以上）|
| **無障礙環境獎勵** | 5% | 超過法定標準設計 |
| **開放空間獎勵** | 5% | 提供公共開放空間 |
| **綠能設施獎勵** | 5% | 設置太陽能等綠能設備 |

**法定總上限：基準容積的 50%**（依都市更新條例第65條）

### 💡 專業建議

#### 獎勵組合策略

**推薦組合（高CP值）：**
1. 基本獎勵（20-25%）
2. 綠建築獎勵（10%）
3. 智慧建築獎勵（10%）
4. 無障礙獎勵（5%）
**預估總獎勵：45-50%**

**成本效益分析：**
```
假設基準容積 300%
獎勵 50% → 總容積 450%

成本投入：
- 綠建築升級：約增加 3-5% 工程成本
- 智慧建築系統：約增加 2-3% 工程成本
- 容積價值：增加 50% 可售面積

投資報酬率：約 10-15 倍
```

#### 申請流程

1. **設計階段**（建築師主導）
   - 規劃綠建築設計（通風、採光、節能）
   - 整合智慧建築系統（管理、安全、通訊）
   - 預留無障礙設施空間

2. **送審階段**（都更處審議）
   - 提交事業計畫書時一併申請
   - 準備綠建築/智慧建築候選證書
   - 經都市更新審議會審查通過

3. **驗收階段**（取得正式標章）
   - 完工後申請正式綠建築標章
   - 申請正式智慧建築標章
   - 標章取得後才能取得使用執照

### ⚠️ 風險提醒

1. **獎勵非保證取得**
   - 需經審議會審查，可能打折或不予獎勵
   - 建議預留彈性，不要完全依賴最高獎勵

2. **標章取得條件**
   - 綠建築：需符合日常節能、水資源等9大指標
   - 智慧建築：需建置資訊通信、系統整合等8大系統
   - 未達標準將無法取得獎勵

3. **維護成本考量**
   - 智慧建築系統需長期維護更新
   - 綠建築設備（如雨水回收）需定期保養
   - 建議納入社區管理費預算

### 📝 後續步驟

1. 委託專業建築師評估基地條件
2. 試算各項獎勵的成本效益比
3. 決定申請組合後納入事業計畫
4. 設計階段即取得綠建築/智慧建築候選證書
5. 完工後申請正式標章

---

**相關法規：**
- 都市更新條例第65條（容積獎勵上限）
- 都市更新建築容積獎勵辦法第4-8條
- 台北市都市更新自治條例第15條（獎勵項目）
- 綠建築標章申請審核認可及使用作業要點
- 智慧建築標章申請認可評定及使用作業要點
```

## 範例 3：權變 vs 協議比較

### 用戶提問
```
請問權利變換和協議合建有什麼差別？我們社區應該選哪一種比較好？
```

### 預期回應
```markdown
## 權利變換 vs 協議合建完整比較

### 📋 兩種方式差異表

| 比較項目 | 權利變換 (權變) | 協議合建 (協議) |
|---------|---------------|---------------|
| **法源依據** | 都市更新條例第25-1條 | 都市更新條例第25條 |
| **同意門檻** | 劃定區內：1/2<br>非劃定區：3/4 | 需100%同意 |
| **強制力** | 可強制少數戶參加 | 無強制力 |
| **分配方式** | 按權利價值比例分配 | 自由協商約定 |
| **程序複雜度** | 高（需政府核定）| 中（雙方合意即可）|
| **時程** | 3-5年 | 1-3年 |
| **容積獎勵** | 可申請（最高50%）| 可申請（最高50%）|
| **稅賦優惠** | 有（土增稅減免）| 有（土增稅減免）|
| **爭議處理** | 經審議會裁決 | 雙方協商或訴訟 |
| **適用情境** | 難以100%達成共識 | 全體住戶意見一致 |

### ⚖️ 法規適用

#### 權利變換（第25-1條）
**優點：**
- 不需100%同意即可推動
- 公平性較高（專業估價分配）
- 政府介入監督，保障較完整

**缺點：**
- 程序冗長，需多次審議
- 估價專業複雜，易生爭議
- 成本較高（需聘估價師、律師等）

#### 協議合建（第25條）
**優點：**
- 彈性高，條件可協商
- 時程較快，不需政府核定
- 成本較低

**缺點：**
- 需100%同意（一戶反對即無法進行）
- 公平性取決於談判能力
- 易受釘子戶阻撓

### 💡 專業建議

#### 選擇判斷流程圖

```
社區住戶數
    ├─ 10戶以下 → 協議合建（易達成共識）
    └─ 10戶以上
        ├─ 已有90%以上同意 → 可嘗試協議合建
        └─ 同意度70-90% → 建議採權利變換
            └─ 同意度50-70% → 必須採權利變換
```

#### 您的社區適合哪一種？

**建議採用權利變換，如果：**
1. 社區戶數較多（超過20戶）
2. 存在少數反對戶（同意率低於95%）
3. 產權複雜（繼承、共有持分多）
4. 希望有政府監督保障

**建議採用協議合建，如果：**
1. 社區戶數較少（10戶以內）
2. 全體住戶已達成高度共識（95%以上）
3. 希望快速完成更新
4. 已有信任的建商合作對象

### ⚠️ 風險提醒

#### 權利變換風險
1. **時程風險**：政府審議會議可能延宕
2. **估價爭議**：更新前後估價可能有不同意見
3. **成本較高**：專業服務費用（估價、律師、建築師）約佔總成本3-5%

#### 協議合建風險
1. **釘子戶風險**：一戶不同意即全盤皆墨
2. **不公平風險**：弱勢住戶可能被迫接受不利條件
3. **法律風險**：合約內容如有瑕疵，日後易生糾紛

### 📝 實務建議

#### 混合策略（推薦）

許多成功案例採取「先協議、後權變」策略：

**第一階段（協議嘗試期 6-12個月）**
1. 召開社區說明會，爭取最大共識
2. 與建商協商合建條件
3. 募集同意書，目標100%

**第二階段（權變準備期）**
- 如第一階段未能達成100%同意
- 但已達權變門檻（1/2 或 3/4）
- 轉向申請權利變換程序

**優點：**
- 保留協議的彈性與速度
- 有權變作為後盾，增加談判籌碼
- 降低釘子戶阻撓風險

### 📝 後續步驟

1. **評估現況**
   - 統計社區戶數與產權狀況
   - 進行初步意願調查
   - 評估達成100%共識的可能性

2. **專業諮詢**
   - 找3-5家都更顧問公司提案
   - 比較權變/協議的試算結果
   - 評估時程與成本差異

3. **社區決議**
   - 召開區分所有權人會議
   - 投票決定採用方式
   - 委任實施者或成立更新會

4. **啟動程序**
   - 如採協議：與建商簽訂合建契約
   - 如採權變：向政府申請事業計畫

---

**相關法規：**
- 都市更新條例第25條（協議合建）
- 都市更新條例第25-1條（權利變換）
- 都市更新權利變換實施辦法全文
```

## 範例 4：危老 vs 都更比較

### 用戶提問
```
聽說危老重建比都更簡單，請問我們應該選哪一個？兩者有什麼差別？
```

### 預期回應
```markdown
## 危老重建 vs 都市更新完整比較

### 📋 快速比較表

| 比較項目 | 危老重建 | 都市更新 |
|---------|---------|---------|
| **法源** | 危老條例 | 都市更新條例 |
| **適用對象** | 危險或老舊建物 | 任何符合條件建物 |
| **基地規模** | 200㎡以上<br>（或合併相鄰土地達200㎡）| 1000㎡以上<br>（或1000㎡以下但完整街廓）|
| **同意門檻** | **100%**（必須全體同意）| 劃定區內：1/2<br>非劃定區：3/4 |
| **容積獎勵** | 基本10%<br>+時程獎勵（已落日）| 最高50% |
| **建蔽率** | 可放寬至基準×1.3 | 依都更計畫規定 |
| **時程獎勵** | 已於2023年5月到期 | 無時程獎勵 |
| **稅賦優惠** | 地價稅、房屋稅減半徵收 | 地價稅、房屋稅減半徵收 |
| **審議程序** | **免經都更審議會**（僅結構安全審查）| 需經都更審議會 |
| **行政時程** | 3-6個月 | 2-5年 |
| **強制力** | 無（需100%同意）| 有（可強制少數戶）|
| **政府補助** | 有（重建補助、融資利息補助）| 無 |

### ⚖️ 適用條件詳細說明

#### 危老重建適用條件（需符合其一）

1. **經建管單位通知為危險建物**
   - 耐震能力評估結果：危險或需補強
   - 海砂屋
   - 需經專業技師鑑定

2. **屋齡30年以上合法建物**
   - 需有使用執照
   - 違建部分不計入獎勵

3. **未達最小基地規模**
   - 基地面積需達200㎡以上
   - 或合併鄰地達200㎡

#### 都市更新適用條件

1. **符合更新地區劃定標準**
   - 建築物老舊，影響都市發展
   - 建築物窳陋，有礙觀瞻
   - 建築物排列不良，妨礙公共安全

2. **基地規模要求**
   - 原則1000㎡以上
   - 或1000㎡以下但完整街廓

### 💡 選擇建議

#### 選擇危老重建，如果：

✅ **符合條件**
- 建物確實危險或屋齡30年以上
- 基地規模200-1000㎡
- 全體所有權人意見一致

✅ **優先考量**
- 希望快速完成（3-6個月核准）
- 不想經過冗長都更審議程序
- 基地規模較小，不適合都更

⚠️ **注意事項**
- **必須100%同意**（一戶不同意即無法進行）
- 容積獎勵較少（最多10%，時程獎勵已落日）
- 需自費聘請結構技師鑑定

#### 選擇都市更新，如果：

✅ **符合條件**
- 基地規模1000㎡以上
- 無法達成100%同意
- 位於劃定更新地區內

✅ **優先考量**
- 希望爭取最大容積獎勵（可達50%）
- 有少數不同意戶需強制參加
- 願意等待較長審議時程

⚠️ **注意事項**
- 程序複雜，需2-5年
- 成本較高（專業服務費3-5%）
- 需經多次審議會審查

### 💰 財務試算比較

假設條件：
- 基地面積：500㎡
- 原建物：5層樓公寓（20戶）
- 基準容積：300%
- 所有權人：100%同意

#### 方案一：危老重建
```
基準容積：500㎡ × 300% = 1,500㎡
危老獎勵：10%
總容積：1,500㎡ × 1.1 = 1,650㎡

重建後戶數：約25戶
- 原住戶分回：20戶
- 建商分得：5戶

時程：3-6個月審查 + 2年施工 = 2.5年
```

#### 方案二：都市更新
```
基準容積：500㎡ × 300% = 1,500㎡
都更獎勵：50%
總容積：1,500㎡ × 1.5 = 2,250㎡

重建後戶數：約35戶
- 原住戶分回：20戶
- 建商分得：15戶

時程：2-3年審議 + 2年施工 = 4-5年
```

#### 比較結論
```
危老優勢：
- 時程快2-3年
- 程序簡單
- 成本較低

都更優勢：
- 容積多600㎡（約10戶）
- 原住戶分回坪數可更大
- 建商利潤更高，條件可能更好
```

### ⚠️ 常見錯誤迷思

❌ **迷思1：「危老一定比較快」**
✅ **真相：** 如果無法100%同意，危老根本無法啟動。都更雖慢但有強制力。

❌ **迷思2：「都更獎勵比較多，一定比較好」**
✅ **真相：** 獎勵多但時程長2-3年，資金成本與機會成本也較高。

❌ **迷思3：「危老不用審議，一定會過」**
✅ **真相：** 仍需結構安全審查與建管審查，不合規格仍會被退件。

❌ **迷思4：「我的房子30年了，自動可以危老」**
✅ **真相：** 需全體100%同意 + 基地達200㎡ + 有合法建照。

### 📝 實務建議：混合策略

許多成功案例採取「先嘗試危老，不成再轉都更」策略：

**第一階段（危老評估期 3-6個月）**
1. 評估是否符合危老條件
2. 進行100%同意意願調查
3. 初步與建商接洽條件

**第二階段（方案選擇）**
- 如能達成100%同意 → 走危老（快速）
- 如無法100%同意 → 轉都更（有強制力）

**優點：**
- 兩手準備，降低風險
- 危老測試可作為談判基礎
- 未來仍可轉換跑道

### 📝 後續步驟

1. **自我檢測**
   - 確認建物屋齡、結構狀況
   - 評估基地規模
   - 調查所有權人同意意願

2. **專業鑑定**
   - 委託結構技師進行初步評估
   - 確認是否符合危老或都更條件

3. **財務試算**
   - 找2-3家建商提案
   - 比較危老/都更的分配條件
   - 評估時程與財務效益

4. **社區決議**
   - 召開說明會，說明兩種方案差異
   - 投票決定採用方式
   - 啟動後續程序
```
