# Taiwan Urban Renewal Expert Skill

## 都市更新全方位顧問技能包

這是專為 Claude Code 開發的台灣都市更新專家技能，整合了實施者、建築師與法律顧問的專業能力。

### 📦 技能概述

**技能名稱:** taiwan-urban-renewal
**版本:** 1.0.0
**語言:** 繁體中文 (Traditional Chinese - Taiwan)
**領域:** 台灣都市更新法規、建築規劃、法律諮詢

### 🎯 核心功能

1. **法規諮詢** - 《都市更新條例》及相關子法精準解釋
2. **案件評估** - 容積率獎勵、可行性分析、財務評估
3. **策略規劃** - 同意門檻達成、談判策略、風險管理
4. **法律諮商** - 代為拆除、少數不同意戶、信託機制
5. **文書製作** - 事業計畫書、權利變換計畫、合建契約

### 🚀 使用方式

#### 在 Claude Code 中使用

當您詢問台灣都市更新相關問題時，Claude 會自動以都市更新專家的身份回應。

#### 範例問題

```
我們社區在台北市信義區，想推動都更需要多少住戶同意？
```

```
危老重建跟都更有什麼差別？哪一個比較適合我們？
```

```
容積獎勵最高可以拿到多少？需要符合什麼條件？
```

```
請幫我評估這棟建築物是否適合推動都更
```

### ⚖️ 法律聲明

本技能提供的資訊僅供參考，不構成正式法律意見。具體案件建議諮詢：
- 取得執照的律師（法律訴訟）
- 取得執照的建築師（建築設計）
- 合格的估價師（不動產估價）
- 專業的會計師（稅務規劃）

### 📝 技能特色

1. **精準引用法條** - 所有法律解釋都會引用具體條文編號
2. **在地化考量** - 考慮台北市、新北市等地方自治條例差異
3. **結構化輸出** - 使用表格、條列式呈現複雜資訊
4. **實務導向** - 提供可執行的建議與後續步驟
5. **全繁體中文** - 使用台灣專業術語，不含簡體字

### 🎓 適用對象

- **地主/土地所有權人** - 了解權益與選項
- **住戶/建物所有權人** - 評估參與意願
- **實施者/建商** - 規劃執行策略
- **更新會成員** - 推動案件進度
- **建築師/都市計畫師** - 專業技術諮詢
- **律師/法務人員** - 法規適用確認
